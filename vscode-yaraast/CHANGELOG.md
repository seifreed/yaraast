# Changelog

All notable changes to the YARAAST VS Code extension are documented in this file.

## 0.1.0

- Added a real language server integration for YARAAST.
- Added diagnostics, hover, completion, definition, references, rename and symbols.
- Added formatting and range formatting support.
- Added multi-dialect support for YARA, YARA-L and YARA-X.
- Added structural refactor previews and authoring code actions.
- Added runtime status, diagnostics and metrics commands.
- Added snippets, grammar, icons and VSIX packaging assets.
